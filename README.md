## Getting Started

### 1. Installation
```bash
npm install
```

### 2. Development
```bash
npm run dev
```

### 3. Build
```bash
npm run build
```

## Structure
*   `src/components`: Reusable UI components.
*   `src/pages`: Route-level page components.
*   `src/layouts`: Layout wrappers (MainLayout, etc).
*   `public/`: Static assets (images, fonts).
